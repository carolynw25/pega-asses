# Setup

### 1. Clone the repository:
```bash
git clone https://github.com/carolynw25/pega-assess.git
cd pega-assess
```

### 2. Install dependencies:
```bash
npm install
```

### 3. Start the server:
```bash
npm run dev
```
Then follow terminal prompt.

> Note: Make sure you are using Node.js v22+
